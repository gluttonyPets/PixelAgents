quiero que planifiques un proyecto en netcore, donde el cliente esté separado del servidor, serán 2 proyectos diferentes, el servidor expone una api y el cliente web pregunta, el concepto de proyecto es hacer varios módulos de ia, con el objetivo de crear contenido para redes sociales, el concepto es que cada módulo tenga una taréa específica así es más facil mejorar a largo plazo, por ejemplo un módulo se va a encargar de buscar temas que le interesa a la gente, otro módulo se encarga de crear plantillas en photosop, otro módulo se encarga de montar las plantillas con datos relevantes del tema que ha propuesto el módulo 1 y por último un módulo que se encarge de definir la mejor fecha y hora para subir la publicación en diferentes plataformas. El concepto de redes sociales es para lo que lo usare unicialmente, pero la idea es que se puedan crear modulos (avatares) según el concepto que se quiera abordar, la idea es representar en el cliente web, como todos los avatares trabajan en concordancia para sacar el producto final, la temática es de un videojuego pixelart en 2d, similar al game dev tycon y las ias deben estar en una oficina, cada una tienes sus skills etc y se representan como personas(avatares)
